# Youssef Tarek — Portfolio

A 3D, animated personal portfolio for **Youssef Tarek**, a Software QA & Cybersecurity Engineer based in Cairo. Built with **React**, **TypeScript**, **Three.js**, **GSAP**, and **Lenis**.

## Tech Stack

- React + TypeScript (Vite)
- Three.js / React Three Fiber (3D character)
- GSAP + ScrollTrigger (scroll animations)
- Lenis (smooth scroll)

## Getting Started

```bash
npm install
npm run dev      # local dev server
npm run build    # production build
```

## Customize

All site content lives in [`src/config.ts`](src/config.ts): identity, about, experience, projects, skills, and contact links. Update that file to change copy. Replace images in `public/images/` and the résumé at `public/Youssef_Tarek_CV.pdf`.
